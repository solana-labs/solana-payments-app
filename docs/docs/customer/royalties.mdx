## Future Features - Royalty Program

In the future, we plan to introduce a royalty program. With this program, customers can earn points on purchases and unlock NFTs after spending certain amounts. These rewards can be used for discounts on future purchases or other exclusive perks. Stay tuned for more details.

**Please note that we do not store any personal customer data, apart from your wallet address.**
